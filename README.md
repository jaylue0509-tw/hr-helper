# HR 人資小幫手 (PureHR)

這是一個專為 HR 設計的輔助工具，包含抽獎功能與分組視覺化工具。使用 React + Vite 開發。

## 功能特色

### 🎲 幸運抽獎 (Lucky Draw)
- 支援名單匯入
- 隨機抽籤動畫
- 歷史紀錄保存

### 📊 分組視覺化 (Group Visualizer)
- 視覺化分組顯示
- 支援拖直觀操作 (如果有的話)

## 專案設定與安裝

本專案使用 Node.js 環境。

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

開啟瀏覽器並訪問顯示的 URL (通常是 `http://localhost:5173`)。

### 建置生產版本

```bash
npm run build
```

## 部署

本專案已設定 GitHub Actions，推送到 `main` 分支時會自動部署至 GitHub Pages。
